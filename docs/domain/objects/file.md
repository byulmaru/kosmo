# File 객체

## 정의

File은 Media가 사용하는 원본 또는 파생 파일 표현이다. 저장소 구현과 공개 URL은 소유하지 않는다.

## 상태

### File Representation Type

| 값       | 의미                          |
| -------- | ----------------------------- |
| Original | 업로드된 원본 파일 표현       |
| Derived  | Original에서 생성된 파생 표현 |

## 속성

| 속성      | 타입/nullability | 검증 정책                       | 존재 조건   | 조회 조건            | 조회 권한 |
| --------- | ---------------- | ------------------------------- | ----------- | -------------------- | --------- |
| MIME Type | 문자열, 필수     | 허용 목록 검증의 대상이다       | 항상        | Media 조회 정책 통과 | 없음      |
| Byte Size | 정수, nullable   | 0보다 크며 측정된 경우 기록한다 | 항상        | Media 조회 정책 통과 | 없음      |
| Width     | 정수, nullable   | 0보다 큰 이미지 폭              | 이미지 File | Media 조회 정책 통과 | 없음      |
| Height    | 정수, nullable   | 0보다 큰 이미지 높이            | 이미지 File | Media 조회 정책 통과 | 없음      |

## 관계

| 관계        | 대상                | 방향          | cardinality | 존재 조건      | 조회 조건            | 조회 권한 |
| ----------- | ------------------- | ------------- | ----------- | -------------- | -------------------- | --------- |
| Media       | [Media](./media.md) | File -> Media | 1 -> 1      | 항상           | Media 조회 정책 통과 | 없음      |
| Source File | [File](./file.md)   | File -> File  | 1 -> 1      | Type이 Derived | Media 조회 정책 통과 | 없음      |

## 행동

| 행동         | 행동 주체 | 대상 객체 | 입력값        | 권한                    | 조건                                            | 결과                                                |
| ------------ | --------- | --------- | ------------- | ----------------------- | ----------------------------------------------- | --------------------------------------------------- |
| Derived 생성 | 시스템    | File      | Original File | `System.ImageProcessor` | 입력 Type이 Original이고 이미지 검증을 통과한다 | Type=Derived인 File이 생성되고 Source 관계가 생긴다 |

Original File은 Media 이미지 업로드 행동 결과로 생성한다.

## 권한

| 권한                    | 종류 | 성립 조건                        |
| ----------------------- | ---- | -------------------------------- |
| `System.ImageProcessor` | 독립 | 시스템이 파생 이미지 처리 주체다 |

## 조회 정책

- File은 연결된 Media 조회 정책을 그대로 따른다.
- 공개 URL은 File 속성이 아니라 Media 조회 결과다.

## 확정 용어

- 파일 표현: File
- 파일 원본: Original File
- 파생 이미지: Derived Image
- 썸네일: Thumbnail

## 제외/보류

- 저장 위치, storage key, 구체 파일 형식과 크기 제한, Hash, EXIF, 파생 이미지 생성 방식은 구현/OpenSpec에서
  다룬다.
